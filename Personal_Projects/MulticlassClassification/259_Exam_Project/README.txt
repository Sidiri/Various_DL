In order to run the system it is necessary to have:
- Robot Operating System (ROS)
- Python 2.7 and Python 3.6
- FastAi
- Packet manager (recommend Conda)
- openni2

To run the system:
1. Terminal 1 - roscore
2. Terminal 2 - roslaunch openni2_launch openni2.launch
3. Terminal 3 - Run ros_script.py

The classifications will be printed in terminal window 3.
