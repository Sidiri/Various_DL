For å kunne kjøre systemet må du ha Linux med følgende programvare:
- Robot Operating System (ROS)
- Python2.7 og Python3.6
- fastai (anbefaler Conda som package manager)
- openni2

For å kjøre systemet: 
1. Terminal 1 - roscore
2. Terminal 2 - roslaunch openni2_launch openni2.launch
3. Terminal 3 - kjør ros_script.py

Predikasjonene vil da skrives ut i terminalvindu 3.
